# Web_Scraping_Python
Teste de raspagem de dados em sites de análise de ações e criptomoedas. 
O objetivo deste código é coletar dados de qualquer site do mercado financeiro utilizando Web Scraping 
de forma automática e atualizada. 
Para testar, foram usados os sites: https://www.fundamentus.com.br/ e https://investidor10.com.br/criptomoedas/. 
O primeiro site disponibiliza informações financeiras e fundamentalistas das empresas com ações listadas na Bovespa, 
o segundo disponibiliza diversas informações sobre as principais criptomoedas. 

Requisitos:
Instalação das bibliotecas Requests, Pandas, Beautiful Soup e Matplotlib.

Instruções de uso:
Dentro da pasta Web_Scraping_Finanças há três outras pastas.
- Execution: Coleta os dados dos Proventos do Banco Inter, monta a tabela, gera alguns gráficos e um arquivo para excel.
- Execution 2: Coleta os dados de quem são os principais acionistas e a sua participação nas ações da empresa Intelbras, 
  monta a tabela e gera um arquivo para excel.
- Execution 3: Coleta diversos dados das principais criptomoedas presentes no mercado, monta uma tabela e gera um arquivo para excel.

Considerações finais: Com algumas simples modificações no código (nomes e números de colunas) 
é possível coletar dados financeiros de muitos sites.
